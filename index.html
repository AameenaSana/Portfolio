To-Do List

This To-Do App is a powerful yet simple task management tool designed to keep you organized and efficient. Easily add, delete, and mark tasks as completed with a strikethrough effect. With new features like one-click task clearance, timers with alarms to keep you on track, and a speech-to-text input for hands-free task creation, managing your to-do list has never been easier. Whether it’s daily errands, work assignments, or personal reminders, this app helps you stay productive with ease and convenience.

Expense Tracker

This MERN stack Expense Tracker is a smart financial management tool that helps users track their total income, expenses, and balance in real time. It provides a detailed transaction history where each expense can be categorized (e.g., "Food," "Rent," "Shopping"), making budgeting easier. The app features interactive graphs to visualize spending patterns and income trends over time. Built with MongoDB, Express.js, React, and Node.js, it ensures secure data storage and a smooth user experience, helping users stay financially organized with ease.

Blog Website

This project is a full-stack blog application built using the MERN stack (MongoDB, Express, React, and Node.js). It allows users to register, log in, and create, edit, delete, and view blog posts. The frontend is built with React for a responsive user interface, while the backend uses Node and Express to handle API requests and MongoDB for data storage. It's a great hands-on project to learn how to connect a frontend with a backend and manage full CRUD operations with authentication.
